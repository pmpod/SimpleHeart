Detailed Description

NumericStrategy is a main interface class for the numerical integration of the model's equations. It provides the interface for numerical schemes like Forward Euler Scheme, modified Backward Euler Scheme, Allexandre&Otani scheme etc.

The classes that implement a concrete numerical calculation strategy should implement this. The XXX AtrialMachine Class uses this to call the concrete numerical scheme to integrate electrophysiological equations of a given model.
